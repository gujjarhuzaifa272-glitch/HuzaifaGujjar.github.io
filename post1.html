How I built a modern landing page

Published November 25, 2025

In this article I’ll walk through the process of crafting a modern landing page using pure HTML, CSS and a bit of JavaScript. The goal was to build a professional, dark‑themed layout that showcases a portfolio, services and contact form—all while keeping the code lightweight and easy to customize.

The first step was to define the structure of the page: a hero section to introduce myself and the value proposition, followed by sections for services, portfolio, a call to action, blog, contact and social links. I chose a dark background with a gradient and used an accent colour to highlight important elements such as buttons and links.

For the portfolio and services, I created reusable cards using CSS grid. Each card floats on a semi‑transparent panel with rounded corners and subtle shadows, giving the page depth without feeling heavy. The contact form uses simple styling and a mailto link so visitors can reach me quickly.

Finally, I added a responsive navigation bar and ensured the layout adapts gracefully on mobile devices. By defining all colours and sizes in CSS variables, it’s easy to tweak the look and feel across the entire site.
